## Tips Aman Saat Menggunakan API Key

Satu hal yang wajib diperhatikan: jangan pernah expose API key di frontend. Ini kesalahan yang masih cukup sering terjadi, terutama saat membuat demo cepat.

API key yang bocor bisa dipakai orang lain dan menghabiskan credit kamu tanpa izin. Untuk menghindarinya, gunakan pendekatan yang lebih aman seperti:

- Simpan API key di environment variable.
- Panggil API dari backend, bukan langsung dari browser.
- Gunakan server-side proxy untuk meneruskan request dari frontend.
- Batasi akses dan pantau penggunaan API secara berkala.

Kalau kamu sedang membangun prototype, tetap biasakan pola integrasi yang aman sejak awal. Kebiasaan kecil ini bisa menyelamatkan project saat nanti mulai dipakai lebih banyak orang.
